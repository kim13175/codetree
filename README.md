# codetree (코딩 연습을 할 수 있는 웹 사이트)

### 코드 트리를 선택한 이유
---
```
기존 python을 이용하여 코딩 테스트 준비를 하였지만 프론트 엔드 개발 분야에서 기본이 되는 javascript
에 대한 숙련도를 높이고자 진행함   
```

### 공부한 각 카테고리 별로 디렉토리를 분류함
---
- string and basic : 문자열 및 기초 카테고리
  - 문자열 슬라이싱, 포매팅 관련 학습
- function : 함수 카테고리
  - call by value, call by reference 학습
  - 전역 변수와 지역 변수 학습
